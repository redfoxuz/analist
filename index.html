1 Pandas — jadval ma'lumotlar (Excel o'rnini bosuvchi super qurol)

CSV/Excel, ma'lumotlar bazasi yoki API dan kelgan jadval ko'rinishidagi ma'lumotni o'qish, tozalash, tahlil qilish va natijani qayta saqlash uchun eng asosiy kutubxona.

Pandas · jadval ma'lumotlar

Asosiy tushunchalar

pandas — bu jadval ma'lumotlar (row/column) bilan ishlash uchun yozilgan yuqori darajadagi kutubxona. Excel faylingizni tasavvur qiling: pandasda bu DataFrame bo'ladi.

  • Series — 1D, indekslangan ustun (masalan faqat yosh ustuni).
  • DataFrame — bir nechta Series dan tashkil topgan 2D jadval.
  • Index — qatorlarni belgilovchi qiymat (0,1,2,... yoki matnli).
  • Label — ustun nomlari: ["ism", "yosh", "bahosi"].

Pandas yordamida siz filtrlash, sortlash, guruhlash, agregatsiya (o'rtacha, min/max, sanoq) va tozalash (NaN, dublikatlar) ishlarini sodda qilib bajarishingiz mumkin.

Tipik ish oqimi (workflow)

  1. Ma'lumotni yuklash: pd.read_csv()
  2. Ko'zdan kechirish: df.head(), df.info()
  3. Tozalash: NaN, xatolar, turlarni to'g'rilash
  4. Tahlil: filtrlash, groupby, pivot, feature engineering
  5. Natijani saqlash yoki vizualizatsiya qilish

2 NumPy — tezkor massivlar va matematik amallar

NumPy — ilmiy hisoblashning yuragi. Pandas, SciPy, scikit-learn va ko'plab kutubxonalar ichida NumPy ndarray massivlaridan foydalanadi.

NumPy · array & vektorlashtirish

Nima uchun NumPy kerak?

Oddiy Python ro'yxatlari (list) moslashuvchan, lekin katta masshtabdagi matematik amallar uchun sekin. NumPy array esa C da yozilgan bo'lib, xotira jihatdan ixcham va juda tez ishlaydi.

  • Vektorlashtirilgan amallar: butun massivga bir yo'la amal qo'llash
  • Broadcasting: o'lchamlari turlicha massivlar bilan hisoblash
  • Ko'p o'lchamli massivlar: 1D, 2D (matritsa), 3D (tensor) va h.k.
  • Matritsa algebra: skalyar, vektor, matritsa amallari
  • Tasodifiy sonlar: np.random

Python list vs NumPy array

import numpy as np

py_list = [1, 2, 3, 4]
np_arr  = np.array([1, 2, 3, 4])

# Listda elementlar ustida aylanish kerak
py_list2 = [x * 2 for x in py_list]

# NumPy da esa butun massivga bir yo'la amal
np_arr2 = np_arr * 2

Vektorlashtirish, broadcasting va chiziqli algebra

Vektorlashtirilgan amallar

# Har bir qatordan 1 ni ayirish

# Ustunlar bo'yicha normalizatsiya qilish

3 Matplotlib — grafiklar va vizualizatsiya asosi

Matplotlib — Python uchun eng qadimiy va juda moslashuvchan chizish kutubxonasi. Ko'pchilik kutubxonalar (Seaborn, pandas.plot, va h.k.) ichida Matplotlib dan foydalanadi.

Matplotlib · vizualizatsiya

Pyplot va obyektga yo'naltirilgan (OO) uslub

Matplotlib bilan ikki xil uslubda ishlash mumkin:

  • Pyplot uslubi — import matplotlib.pyplot as plt va plt.plot(...).
  • OO uslub — fig, ax = plt.subplots(), so'ng ax.plot() va boshqalar.

Kichik loyihalar uchun pyplot yetarli, lekin ko'p grafiklar va murakkab layoutlarda OO uslub ancha qulay.

Oddiy chizma (line plot)

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [1, 4, 9, 16]

plt.figure(figsize=(5, 3))
plt.plot(x, y, marker="o")
plt.title("Oddiy chizma")
plt.xlabel("x")
plt.ylabel("y")
plt.grid(True)
plt.show()

Eng ko'p ishlatiladigan grafiklar

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)

fig, axs = plt.subplots(2, 2, figsize=(7, 5))

# 1. Chiziqli grafik
axs[0, 0].plot(x, y, color="cyan")
axs[0, 0].set_title("Sinus funktsiyasi")

# 2. Nuqtalar (scatter)
axs[0, 1].scatter(x, y, color="magenta", s=10)
axs[0, 1].set_title("Scatter")

# 3. Ustunli grafik (bar)
labels = ["A", "B", "C"]
values = [10, 15, 7]
axs[1, 0].bar(labels, values, color="orange")
axs[1, 0].set_title("Bar chart")

# 4. Histogramma
rand = np.random.randn(500)
axs[1, 1].hist(rand, bins=20, color="lime", alpha=0.7)
axs[1, 1].set_title("Histogram")

plt.tight_layout()
plt.show()

Maslahat:

Har bir grafikda title, xlabel, ylabel, grid, legend kabi qismlarni to'g'ri yozish — grafikni tushunarli va professional qiladi.

Matplotlib bo'yicha tez eslatmalar

Grafik turini tanlang

Chiziqli grafik (line plot)

Eng ko'p ishlatiladigan grafik turi. Vaqt bo'yicha o'zgarish, trend va ketma-ketliklarni ko'rsatish uchun juda mos.

Kod namunasi:

4 Seaborn — statistik grafiklar va chiroyli dizayn

Seaborn — Matplotlib ustiga qurilgan yuqori darajadagi vizualizatsiya kutubxonasi. U avtomatik ravishda chiroyli stil, rang palitralari va statistik grafiklarni taqdim etadi.

Seaborn · statistik vizualizatsiya

Asosiy afzalliklari

  • Chiroyli default stil (ranglar, grid, fontlar)
  • Pandas DataFrame bilan juda qulay ishlaydi
  • Statistik grafiklar: distribution, regression, categorical
  • Bir nechta o'zgaruvchilar orasidagi bog'lanishni tez ko'rish (pairplot, jointplot)

Tez start: Seaborn + misollar datasetlari

import seaborn as sns
import matplotlib.pyplot as plt

# Tayyor datasetlardan birini olish
tips = sns.load_dataset("tips")

sns.set_theme(style="darkgrid")
plt.figure(figsize=(6, 4))

# Scatter + regressiya chizig'i
sns.regplot(data=tips, x="total_bill", y="tip")

plt.title("Hisob summasi va choychaqa orasidagi bog'lanish")
plt.show()

Asosiy grafik turlari

import seaborn as sns
import matplotlib.pyplot as plt

penguins = sns.load_dataset("penguins")

# 1. Tarqatish (distribution): histplot / kdeplot
plt.figure(figsize=(5, 3))
sns.histplot(data=penguins, x="flipper_length_mm", kde=True)
plt.title("Qanot uzunligi tarqatilishi")
plt.show()

# 2. Kategoriyalar: boxplot / violinplot
plt.figure(figsize=(5, 3))
sns.boxplot(data=penguins, x="species", y="body_mass_g")
plt.title("Tur bo'yicha tana massasi")
plt.show()

# 3. Bog'lanish: scatterplot / pairplot
sns.pairplot(penguins, hue="species")
plt.show()
